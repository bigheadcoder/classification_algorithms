Most commonly used Classification algorithms in Machine learning.

Keerti Prajapati

May 5, 2020·6 min read

Classification is a supervised Machine learning approach.

Before we learn about what classification is? , let’s just differentiate both the approaches of supervised learning. Following diagram explains it

Classification is a technique of categorizing /classifying given data point into classes.

For instance,

1. whether the given mail is spam or not.

2. Predicting whether monsoon will be normal next year or not.

Following are four most commonly used classification:

KNN(K Nearest Neighbour)

  1. KNN is simple,easy to understand supervised Machine learning algorithm.
  2. KNN used for classification as well as regressions problem ,but it is widely used in classification.
  3. KNN is a lazy learner because , there is no training phase . It requires training data points during classification.
  4. KNN is non-parametric learning algorithm because it does assume anything about underlying data.
  5. Let’s understand how KNN works:
  • In KNN algorithm ,we first determine value of k i.e number of neighbours to be considered .
  • Calculate distance between training data points and new data point and consider only k data points with minimum distance. Distance can be calculated using either Euclidean or Manhattan.
  • Assign the new data point to class with higher number of votes.

Above figure contains two classes namely class A(yellow circle) and class B(purple circle).

Now ,we have to decide to which class our new data(orange circle) belongs to.

In KNN algorithm we first determines the value of k. In above figure , first we consider k=3. It means we are considering only three nearest data point for classification.

When we use 3 Nearest Neighbour , we have two purple circle and one yellow circle. In this case , KNN algorithm with assign orange circle to class B because class B have higher number of vote than class A . Similarly, when we use k=6 then our new data point will assign to class A.

Now , we can conclude that the decision of KNN is totally dependent on value of k and there is no particular way to determine the best value for "k", so we need to try some values to find the best out of them. This is one of the major drawback of KNN .

Disadvantages of KNN:

  • Always need to determine value of k , which is iterative and sometimes very complex.
  • Computational cost is high because we need to calculate distance between every point.

Decision Tree:

  1. Decision tree is used in classification as well as in regression.
  2. In classification,Decision tree is tree structured classifier .This tree structured classifier helps to make decisions.
  3. Decision tree contains number of nodes . Top node of tree is called root node. Internal node represents feature (or attributes).Leaf nodes represents the outcome or class. There is no further branches after leaf node.
  4. Let’s understand working of decision tree with the help of an example:

Let’s consider we have dataset of people . Dataset contains attributes like age,name, excercise,eating pizzas.

Above figure is a decision tree of our dataset. Let’s assume we’ve got information of new person and we have to decide whether the person is fit or not?

We will give this record to decision tree. Now, decision tree will compare the age attribute of person with the given condition. If the age < 30 then it will compare only ‘eating pizza ’ attribute of person’s record. If the result is Yes then decision tree will classify the person as fit otherwise not fit.

It is very easy to understand the Decision tree ,but the problem arises during building decision tree.

How to build decision tree?

  • Decision tree are build using recursive partitioning to classify data.
  • Important thing to consider in building decision tree is to determine best attribute to split dataset.
  • It is very crucial to determine attribute which is more predective. More predective attribute results into less impurity and lower entropy in leaf node.
  • Pure leaf node results into best model.
  • Impurity of leaf is determined by entropy . Entropy is a measure of randomness . Entropy of node is determined by homogeneity of sample in node . Homogeneous sample means zero entropy.
  • Following steps is used to build the decision tree
  • STEP 1:

Calculate the Entropy attribute . Split the dataset using using that attribute .Again calculate the Entropy after splitting.

  • STEP 2:

Now calculate the information gain. Information gain is the information that can increase the level of certainty.

Information gain = Entropy before split -Entropy after split

  • STEP 3:

Repeat STEP 1 and STEP 2 for all attributes.

  • STEP 4:

Now select attribute with highest Information gain .

Disadvantages of Decision tree:

  • Decision tree works well for categorical data
  • Decision tree do not works well if you have smooth boundaries.

Logistic Regression:

  • Logistic regression is a probability algorithm for categorical variables
  • It is analogous to linear regression ,but it predicts the categorical or discrete variable.
  • Decision boundary of logistic regression is line,plane or hyperplane.
  • Logistic regression also uses linear equation with independent variables to predict target . Predicted value can be anywhere between negative infinity to positive infinity. To overcome this problem we use function which squash the predicted value into either Yes (0) or No(1). This function is called sigmoid function.

Sigmoid function:

Cost function :

  • Cost function of linear regression cannot be applied in logistic regression.
  • Thus, we use logarithmic loss function to calculate cost.

We are not getting into the depth of this concept, because we have software packages that can perform all these task easily.

When to use logistic regression:

  1. If your target data is binary
  2. If you need probabilistic results.
  3. When you need a linear decision boundary.
  4. If you need to understand the impact of a feature.

Support Vector Machine

  • Support vector machine are supervised algorithm that perform classification by finding a separator.
  • Support vector machine performs classification in to steps:
  1. It first maps the data into higher dimensions space so that data can be categorised ,even if data is non linear.
  2. Then finds the separator. Transformation must perform in such a way that separator could be drawn as hyperplane.
  • The process of mapping data into higher dimensions space is called as kernelling. The mathematical functions used for kernelling is called kernel functions. Kernel functions can be of following types:
  1. Linear
  2. Polynomial
  3. RBF
  4. Sigmoid.

As shown in above diagram, hyperplane with maximum margin is optimal hyperplane.

As shown in above diagram , data points which is nearer to margins is called as support vector.

Support vector machines uses these support vector for classification hence SVM are memory efficient.

Disadvantages:

  • Prone to overfitting
  • No probability estimation.
